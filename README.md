# California Housing Data Analysis

## Objective
Analyze the California housing dataset to understand factors influencing median house prices.

## Dataset
California Housing Dataset (CSV)

## Analysis Performed
- Data loading and exploration
- Handling basic data issues
- Exploratory Data Analysis (EDA)
- Data visualization using matplotlib

## Key Insights
- House prices are right-skewed.
- Population does not show a direct linear relationship with house prices.
- Location and housing age influence pricing trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
